Text Conventions

This manual follows the below mentioned documentation conventions.

Keyboard strokes

Keyboard strokes are denoted in bold, underlined letters using Helvetica font. For example, the Tab key refers to the key labeled "Tab" which is usually located to the left of the keyboard, near the "Q" alphabet key.

Controls

Controls like buttons, check boxes, option buttons etc. are denoted in bold letters using purple coloured Helvetica font. For example, the Converter button, means the button labeled "Converter".

File System paths

Individual files, folders and paths to files or folders are presented in green coloured TeleType mode. For example, /boot/beos means the folder named "beos" inside the folder "/boot". Always bear in mind that all paths in the BeOS File System are case sensitive.

External applications

External applications are denoted in italic letters using deep-red coloured Helvetica font. For example, Tracker refers to the application named "Tracker".

Links

As you must have noticed, hyperlinks are shown in underlined medium-blue coloured text, for example: Sofist India.
