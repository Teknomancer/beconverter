Installation

Requirements

To install BeConverter and use it properly you need to atleast have the minimum requirements mentioned below:
  • BeOS R5 or higher.
  • An Intel Pentium (or compatible) based PC
  • 32 MB RAM
  • 3 MB of free hard disk space on a BeOS File System
BeConverter will not work under PowerPC architecture.

Installing BeConverter

BeConverter is distributed in a SoftwareValet package file. Just locate the file BeConverter.pkg and double click it. Now BeConverter's installer will begin. Follow the instructions on screen and complete the installation.

If you wish to move BeConverter to another location, all of its application and data files are self-contained in the BeConverter folder. You are welcome to move the BeConverter folder to any BeOS File System as long as you keep the data files within it.

Important note
No matter to which folder you install or move BeConverter, that folder must be inside a BeOS File System (BFS volume). This is because the critical data files that accompany BeConverter need specific features of BFS.

Hence you cannot install BeConverter to a FAT16 or FAT32 partition.

Uninstalling BeConverter

To complete remove BeConverter from your system follow the steps given below:
  • Drag the BeConverter folder to the Trash.
  • Drag the ~/config/settings/BeConverter folder to the Trash.
  • Drag the 2 templates from /boot/home/config/settings/Tracker/Tracker New Templates to the Trash.
  • Empty the Trash.
Another easy method to remove BeConverter from your system is click "Find..." from the Be menu. Then search for "BeConverter", from the query results that appear simply drag select BeConverter's files and move them to the Trash. Then empty the Trash.
