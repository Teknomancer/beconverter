Using BeConverter

Using the BeConverter is very simple and straight-forward. When you launch BeConverter, the first screen looks something like this:

The various buttons

The Converter... button opens the Converter window through which you perform conversions.

The Options... button brings up the permanent settings window through which you specify some settings for BeConverter.

The About... button brings up BeConverter's copyright and developer information.

The Help... button opens up your browser with this user's guide.

The Close button closes BeConverter and all its open windows.

To perform conversion you have to highlight any category and click the Converter... button or highlight any one category and press Enter or double click the category.

Once you have clicked the button, the BeConverter displays a new window with all the units in the selected (highlighted) category. This new window is called the Converter window. It is through this window you perform the conversions. In our example below we have opened the "Digital Quantity" converter:

As you can see the above illustration explains the various elements of the Converter window's interface.

Just as Tracker reserves a window for each folder you open, BeConverter reserves a Converter window for each category you choose. Thus, you can open and work with several Converter windows simultaneously. This is possible thanks to BeConverter's accomodative framework.

The basics

To use the converter, first you need fill in the Initial value textbox, which is the quantity in the selected base unit. In our example, the initial value is 640 and the selected base unit is "megabyte", which means the value is 640 megabytes.

The Initial value textbox allows you to type in numbers either in the standard normal notation or in the scientific notation. In either case you are allowed to use commas to seperate digits in any manner, hence entering large numbers is made easier. For example, the value of 10 million (10000000) can be entered as 10,000,000 and will be properly understood by BeConverter. Commas can be used anywhere in the number as they will be ignored during calculations.

BeConverter also understands numbers in scientific notation. For example, you can enter 10000 as 1e+4.

Another built-in functionality of BeConverter is that in the Initial value textbox you can type regular mathematical expressions and have them evaluated by pressing Enter. Click here to find out more on this expression evaluator.

As you type in the number or as you choose the base-unit, the converter updates itself by re-calculating all the other units. This is what is called "live" updation. Although it re-calculates all the units "live" using a conversion engine there isn't any slowdown in the speed even on low-end systems.

These are the basic and fundamental steps involved in using the BeConverter.

The extras

Apart from performing basic conversions you will sometimes need to increase or decrease the precision (the number of decimal places) being displayed or you may even wish to see the values in scientific notation.

The .# e checkbox signifies the display notation currently used. You can toggle between scientific notation and the usual numeric notation at anytime. Scientific notation expresses very large or very small values in powers of 10. For example, the notation 999e+3 means 999 × 10³.

The #. 6 button indicates the current precision of the values being displayed. You can choose from a list of precision available (ranges from 0 to 15) by clicking the menu. If you want a bigger precision you must choose so from the Options window.

Important note:
Please bear in mind that the aforementioned options in the Converter window are purely temporary, provided for quick alterations of settings.

They always revert back to the values specified in the Options window, which is where you specify permanent settings. These are only temporary. To see how to set permanent settings for BeConverter refer the "Setting options" section of this manual.

The Copy button isn't the usual copy function. Its an extended copy function that copies the current conversion with all the units in a neat format.

If you wish to copy the results of your conversion to some other program then copying each unit's value one-by-one can be tedious and frustrating. This copy function copies either the selected values to the clipboard or all the values (if none are selected) to the clipboard quickly.

By using the copy button in our example,the following text was copied to the clipboard:

640 megabyte

	= 0.000640 TB
	= 0.640000 GB
	= 640.000000 MB
	= 655360.000000 KB
	= 671088640.000000 B
	= 5368709120.000000 b
If you don't wish to copy all the units to the clipboard you can select a particular value and press C to copy the values from the textboxes of your choice.

The Close button closes the current Converter window.
