BeConverter's history

This page contains a brief history of the changes that have come across the BeConverter.

Version 1.3
(Released)

  • Fixed selection bugs in text-labels
  • Optmized memory handling of text-labels
  • Fixed minor memory leak in ConverterView
  • Typing a character now selects the 1st category starting with that character
  • Fixed some BMessage ownership leaks

Version 1.2
(Released)

  • Added one category: Euro
  • Changed colors for the main list. Selected items look more clear
  • Cleaned up some parts of the code, lots more left
  • Optimized factor-based conversion routine marginally
  • Text-controls now are double-buffered for flicker free draws
  • Multi-selectable text controls
  • Copy function extended to suit selected values
  • Change of License agreement

Version 1.1
(Released)

  • Added double-click action for opening categories
  • Rewrote the Precision button as a menu
  • Made a few other minor optimizations in the evaluator
  • Added Escape shortcut to close Converter window
  • Made values of gigabyte and megabyte more accurate

Version 1.0 - Freeware
(Released)

  • First public release.
  • Added one new category: Fuel consumption.
  • Fixed multi-threading problems. Spawned threads now are slayed.
  • Compacted and optimized the code. Works about 10% faster.
  • Removed hard-coded paths before this release.

Version 1.0 - Final Analysis

  • Error handling for Copy function added.
  • Fixed minor loop-holes in the parsing algorithm

Version 1.0 - Pre-release 5

  • Added functions to register Mime types at start-up
  • Added an option whether or not to register Mime types
  • Re-used a lot of variables. Speed increase by about 2%
  • Error is now flaged if the unit types are in a mixed state
  • Fixed bug in handling the "ceil" and "floor" functions
  • Added some extra exceptions in libBeConverter.so
  • Added one extra function - "abs"
  • Improved parsing speed by removing some unwanted code
  • Added binary, octal, hexadecimal number systems to libBeConverter.so
  • Added precision settings to libBeConverter.so
  • Fixed bug that caused the "abs" function to return absurd values
  • Refined the Converter window GUI a bit
  • Arranged the controls in the code properly to suit the "Tab" navigation
  • Memory leak in Version 1.0 Gamma is finally fixed
  • Added "AboutRequested" to show 'About' when external apps ask for it
  • Reduced binary size by about 6 KB by removing unwanted code
  • Added a few more operators (binary shifts, ands and ors).
  • Added inverse trigonometric and logarithmic functions.
  • Added skin support, pictures are loaded from resource file
  • Fixed a minor bug that caused the expression to get corrupted.
  • BeConverter now restores scroll-bar positions (over looked this before)
  • Removed initiations of multiple instances of a Converter window
  • Fixed a rare bug that caused a failure to load any Converter windows

Version 1.0 - Pre-release 4

  • Fixed a few memory related bugs
  • Added the BeConverter's formula based unit
  • Added one new category: Temperature (at last!)
  • Wrote an algorithm using formula based conversions
  • Fixed a few bugs in the algorithm
  • Added a few more functions like abs, ceil and floor
  • At last fixed the 'Window not locked' warning/error

Version 1.0 - Pre-release 3

  • Major advancement: Completed libBeConverter.so (lot of work!)
  • Now users can type in expressions and have them evaluated
  • C style operator precedence

Version 1.0 - Pre-release 2

  • Added one new category: Digital quantity
  • Fixed bug that crashed app when no. of units were more than 23
  • Added a few more units to the Velocity category

Version 1.0 - Pre-release 1

  • Made some minor adjustments to the GUI
  • Close button also saves the window position
  • Added Precision and Notation options in the Converter window
  • Fixed a practically improbable bug
  • Handled some possible file handling exceptions
  • Remembers window location for all Converter windows
  • Too many bugs in libBeConverter.so. Began complete re-write

Version 1.0 - Gamma

  • Improved speed a little
  • Used more BeOS native classes and structures
  • Began work on the shared library libBeConverter.so
  • Conventionalised the code
  • Multi-threading problems discovered... not yet fixed
  • More memory leaks were discovered... not yet fixed

Version 1.0 - Beta

  • Not released
  • Added four new categories: Torque, Coverage, Magnetic Flux, Luminance
  • Added the "Copy" function to copy all units to the clipboard
  • Fixed a major segment violation
  • Fixed other minor bugs

Version 1.0 - Alpha

  • Not released
  • Added six new categories: Current, Charge, Illuminance, Viscosity, Specific Heat, Voltage
  • Removed the BListView and implemented dynamic option buttons (lot of work!)
  • Loaded the categories, units from BFS as tables and records, rather than from a file
  • Automatically brought in extensive customizability
  • Complete re-write of the converter with around 80% reused code
  • Drastically reduced the output binary size
  • Improved speed about 15 to 20%

Version 1.0 - Debug

  • First build. Lots of bugs... ;-)
  • Came with 20 different in-built categories
  • Supported "live" updations to the converter which made for quick conversion
  • Saves the categories, units in files and not as a database-like structure
  • Binary size was exteremely large due to lot of unwanted code
  • Program was too slow in its execution
  • No customizations were possible
  • "Window not locked" warning/error was found, not fixed
